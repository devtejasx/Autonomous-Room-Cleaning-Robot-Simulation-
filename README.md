This project focuses on developing a simulated mobile robot that can autonomously navigate and cover a room for cleaning purposes. The robot will move through the environment while detecting and avoiding obstacles using simulated sensors. The system will use ROS for robot control and Gazebo for simulation. The robot follows a simple navigation strategy to cover as much floor area as possible, similar to how robotic vacuum cleaners operate.

Objectives

Develop a simulation of a mobile robot capable of autonomous movement in an indoor environment.

Implement obstacle detection using simulated sensors such as LiDAR or distance sensors.

Design an obstacle avoidance mechanism to prevent collisions while the robot moves.

Implement a coverage strategy that allows the robot to move across different parts of the room.

Integrate robot control and sensor data using ROS nodes and communication.

Expected Outcome

A simulated robot that can move autonomously within a room environment.

The robot will be able to detect obstacles and change its path to avoid collisions.

The system will demonstrate continuous movement to cover the floor area effectively.

The project will showcase basic autonomous navigation and obstacle avoidance in a robotic simulation environment using ROS and Gazebo.
